HTML Tags
Div:
A container element used to group other elements, useful for applying styles or layouts.
body:
Encloses the visible content of the web page, such as text, images, and links.
a:
Defines a hyperlink, allowing users to navigate to other web pages or sections of the same page.
p:
Defines a paragraph of text.
img:
Embeds an image in the webpage.
